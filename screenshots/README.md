# Screenshots

This folder is reserved for privacy safe screenshots from the Power BI portfolio version of the project.

## Screenshot rules

Only include screenshots created from:
- synthetic sample data
- fully sanitized demo data

Never include screenshots that display:
- real transaction descriptions
- real counterparties or merchants tied to a real person
- account numbers or payment references
- exact personal balances or sensitive financial details

## Suggested screenshots

Examples:
- model overview
- budget versus actual dashboard
- category breakdown page
- diagnostics or validation page
