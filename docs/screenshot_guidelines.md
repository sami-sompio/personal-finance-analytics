# Screenshot Guidelines

## Purpose

All screenshots included in this repository must be safe for public release and must not expose private financial data.

## Checklist

Before publishing a screenshot verify that it does NOT contain:

- real bank names
- real merchant or counterparty names
- account numbers
- transaction descriptions copied from real data
- identifiable dates tied to real life events
- hidden tooltip pages exposing raw data
- drillthrough tables with detailed transactions
- Power Query previews showing source files
- local file paths

## Recommended Screenshot Types

Prefer screenshots that show:

- dashboard overview pages
- category-level summaries
- KPI cards
- budget vs actual comparisons
- diagnostic visuals

## Avoid

Avoid screenshots that show:

- raw transaction tables
- full transaction logs
- Power Query source previews
- private source file names

## Best Practice

The safest approach is to rebuild dashboard pages using synthetic sample data and create screenshots from that version.
