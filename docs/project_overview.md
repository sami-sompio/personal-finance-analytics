# Project Overview

## Objective

Personal Finance Analytics is an end-to-end analytics project demonstrating how raw financial transactions can be transformed into a structured reporting solution for budgeting and expense analysis.

## Business Goal

The goal of the project is to create a repeatable workflow for:

- consolidating transaction exports
- standardizing raw data
- categorizing transactions
- comparing actual spending against budget
- monitoring budget performance over time
- improving financial visibility

## Technical Scope

The project includes:

- Python-based ingestion and transformation
- rule-based transaction categorization
- combined analytical dataset generation
- Power BI star schema modeling
- DAX-based budget analytics
- dashboards for overview and diagnostics

## Public Repository Version

Because the original project contains sensitive personal financial data, the public repository is designed around:

- synthetic example data
- sanitized documentation
- reviewed screenshots
- example rule files
- privacy-first publishing standards
